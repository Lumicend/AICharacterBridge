# AI Character Bridge

[![English](https://img.shields.io/badge/Language-English-red.svg)](https://github.com/Lumicend/AICharacterBridge/blob/master/README.md)
[![日本語](https://img.shields.io/badge/Language-日本語-blue.svg)](https://github.com/Lumicend/AICharacterBridge/blob/master/README.ja.md)

> **Proof-of-Concept Plugin** | Exclusively for Koikatsu (コイカツ！)

A BepInEx plugin that connects a local LLM (Large Language Model) to game characters in Koikatsu, enabling natural, context-aware conversations. This is a personal proof-of-concept exploring whether AI and games can be bridged together.

This plugin is not compatible with any game title other than "Koikatsu!".

## Key Features

- **Character Card Support**: Define character personalities using JSON files in the widely-used [Character Card V2](https://github.com/malfoyslastname/character-card-spec-v2) / [V3](https://github.com/kwaroran/character-card-spec-v3) format. Cards created with external tools can be used as-is.
- **Context-Aware Conversation**: Past TalkScene conversation logs are automatically saved and carried over into future conversations.
- **In-Game Action Integration**: Depending on the AI's response, in-game actions such as "have lunch together," "go home together," or "make a date reservation" can be triggered.
- **Heroine Parameter Reflection**: The AI's evaluation of the conversation (its judged impression) is automatically reflected in favorability, intimacy, and other parameters.
